## Project Overview

SkillGap AI is a full-stack web application designed to help students evaluate their current skills, identify knowledge gaps, and receive AI-generated recommendations to improve their placement readiness.

The platform combines **React**, **Spring Boot**, **MongoDB**, and **Gemini AI** to deliver an intelligent and interactive learning experience.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/sravya132/skillgap-ai.git
cd skillgap-ai
```

---

### Backend Setup

```bash
cd backend
```

Install dependencies and run:

```bash
./mvnw spring-boot:run
```

or

```bash
mvn spring-boot:run
```

---

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## Configuration

Create an `application.properties` file using the provided `application.properties.example`.

Update the following values:

- Database URL
- Database Username
- Database Password
- Gemini API Key

> API keys and credentials are intentionally excluded from this repository for security reasons.
